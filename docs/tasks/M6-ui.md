# M6 — Minimal UI

## Goal

Create a minimal web UI.

UI must include:

- Input box
- Mode toggle (interactive / deep)
- Streaming output
- Clickable citations

---

## Dependencies

Requires:

- M3 completed
- M3b completed
- M4 completed
- M5 completed

---

## Acceptance Criteria

1. UI loads locally
2. Chat works end-to-end
3. Toggle switches retrieval profile
4. Deep mode visibly returns larger or richer responses

---

## Non-Goals

- No styling polish
- No auth
